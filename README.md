# Map-View-ND
An iOS app that allows you to drop pins on the map and fetch images from Flickr's API

## Technologies Used:
CollectionView <br>
CoreData <br>
SQLLite <br>
MapKit <br>
MapKitAnnotations <br>

## Frameworks Used:
FlickrAPI <br>
