# Map-View-ND
An iOS app that allows you to login via Udacity API and populate students information on the map

## Technologies Used:
TableView <br>
MapKit <br>
MapKitAnnotations <br>

## Frameworks Used:
UdacityAPI <br>
Udacity Parse API <br>
